# CryptoBox Bot

Autonomous Telegram bot posting scheduled crypto price snapshots, simple trading signals, and 1–2 curated news headlines to a channel, with private admin controls.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
